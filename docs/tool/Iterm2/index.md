# iterm2

<PageInfo/>

```Iterm2``` 是苹果系统增加的命令行工具，定制化很强使用方便，并且外观也好看。

## 安装

直接从官网下载安装即可：

<CustomLink title="官网地址" href="https://iterm2.com/"/>

<CustomLink title="官网风格地址" href="https://iterm2colorschemes.com/"/>

或者你可以直接使用 Homebrew 进行安装：

```bash
brew cask install iterm2
```
